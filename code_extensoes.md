### VSCode Browser Sync

É uma espécie de Live Server

Sincroniza uma página HTML no Browser

```console
CTRL + SHIFT + P
```

#### Iniciar servidor

```console
> Browser Sync: Server mode at side panel
```

#### Parar servidor

```console
> Browser Sync: Exit All
```

Abra o Browser na porta 3000

### Live Server

#### Iniciar servidor

```console
ALT + L, ALT + O
```

ou

```console
CTRL + SHIFT + P
```

```console
Live Server: Open with Live Server
```

#### Parar servidor

```console
ALT + L, ALT + C
```

ou

```console
CTRL + SHIFT + P
```

```console
Live Server: Stop Live Server
```
